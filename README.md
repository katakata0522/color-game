# 色の見え方体験ゲーム

色の違いを見分けるブラウザゲームです。通常表示に加え、プロタン・デュータン・トリタンの見え方を簡易的に模した表示と、色差を小さくする強調体験モードを提供します。

## 重要な注意

このコンテンツは娯楽・体験用です。

- 医療上の検査、診断、判定、スクリーニングには使用できません
- ゲームの正誤やスコアから、色覚特性の有無を判断できません
- 画面の色域、輝度、ブルーライト設定、周囲の照明、視聴距離、疲労などで結果が変わります
- 日常生活で色の見分けに継続的な困りごとがある場合は、ゲーム結果ではなく眼科などの医療機関へ相談してください

## 色表示の限界

`script.js`内の色変換行列と序盤ステージの色は、ブラウザ上で違いを簡易的に体験するための近似です。個人ごとの見え方を再現するものではなく、医療機器や標準化された色覚検査の代替ではありません。

行列や出題色を変更する場合は、出典・対象条件・再現限界を確認し、正解不能な問題が生まれないことを自動テストと実機確認で検証してください。

## 主な機能

- 15問の色見分けゲーム
- 通常・プロタン・デュータン・トリタンの簡易表示切替
- 強調体験モード
- キーボード操作
- 効果音ON／OFF
- OS設定へ追従するライト／ダークモード

## 保存データ

テーマ設定だけをlocalStorageへ保存します。

- キー: `colorGameTheme`
- 値: `light`または`dark`
